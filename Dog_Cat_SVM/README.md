# Cats vs Dogs Classification using Support Vector Machine (SVM)

## Objective

Build an SVM classifier to distinguish between images of cats and dogs.

## Dataset

Kaggle Dogs vs. Cats Dataset

## Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

## Steps

1. Load images
2. Resize to 64×64 pixels
3. Flatten image into feature vectors
4. Split into training and testing sets
5. Train an SVM classifier
6. Evaluate accuracy and display predictions

## Run

```bash
python svm_cat_dog_classifier.py
```
